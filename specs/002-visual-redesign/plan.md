# Implementation Plan: Visual Redesign — Premium Personal Site

**Branch**: `002-visual-redesign` | **Date**: 2026-04-19 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `specs/002-visual-redesign/spec.md`

## Summary

Elevate the visual quality of the existing personal site (Bio/Skills/Projects/Contact SPA) from a clean default implementation to a premium minimalist portfolio. Changes span the full stack of presentation: brand identity (MW monogram favicon), typography (Fraunces display serif + Inter body), hero depth (accent gradient), active navigation (IntersectionObserver hook), polished project cards (hover micro-interactions), improved skills grid, and a skills data update (Postman in; Java, Karate, Dredd out). All enhancements preserve the existing minimalist character, WCAG 2.1 AA compliance, and Lighthouse Performance ≥ 90. No structural changes to the SPA or its four sections.
